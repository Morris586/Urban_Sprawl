Help Page

Welcome to the Nairobi Urban Sprawl System. This page will guide you on how to use the site.

Navigation Guide

  • Home: View urban growth and deforestation maps.
  • Pollution Data: View and export pollution data.
  • Deforestation Data: View and export deforestation data.
  • Urban Sprawl Data: View and export urban sprawl data.
  • Summary: View summary statistics for urban sprawl and deforestation.
  • Help: Instructions on using the system.

Using the Data Pages

Each data page allows you to:

  • Search by name.
  • Filter by year.
  • Export the displayed data to a CSV file using the export links in the navbar.

Common Tips

  • If the page shows no results, check your search terms or clear the year filter.
  • You can navigate between pages using the navigation bar at the top.

Need More Help?

Contact the system administrator for support.
